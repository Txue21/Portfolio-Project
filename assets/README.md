# Assets Folder

This folder is for storing images, icons, and other media files.

## Recommended Structure

```
assets/
├── images/
│   ├── projects/          # Project screenshots
│   ├── profile.jpg        # Your profile photo (optional)
│   └── favicon.ico        # Site favicon
└── README.md              # This file
```
